Commands list:

+ help:
    List commands

+ status:
    Is current dir a valid asengine project.
  
+ generate-project <project name>:
    Generate project.

+ generate-component <component name>:
    Generate new component.

+ generate-system <system name>:
    Generate new system.

+ generate-resource <resource name>:
    Generate new resource.

+ generate-module <module name>:
    Generate new module.

+ import-module <module name>:
    Import Module from modules library to your project

+ run:
    Same as build for linux in debug.

+ build <windows | linux | android> <debug | release>:
    Build current project.
